# Linear Regression Project – Predicting Yearly Amount Spent 🛍️

This project uses a **Linear Regression** model to predict how much a customer will spend yearly, based on features like session time, app usage, and website usage.  
This is part of my hands-on learning from a Machine Learning course.

---

## 📊 Dataset Overview

The dataset represents customer behavior for an e-commerce company based in New York City.  
It includes the following features:

- **Avg. Session Length** – Average session time on the website
- **Time on App** – Time spent on the mobile app
- **Time on Website** – Time spent on the website
- **Length of Membership** – Years as a customer
- **Yearly Amount Spent** – (Target) How much the customer spends in a year
